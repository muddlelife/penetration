# Cewl

Cewl是一款采用Ruby开发的应用程序，给它爬虫指定url地址和爬取深度，还可以添加额外的外部连接，接下来Cewl会返回一个字典文件，可以把字典用到一些密码破解工具中，支持命令行

## 选项参数
  
```shell    
1.-w 保存生成字典文件的路径
    
2.-m 生成指定长度的密码字典 如 -m 9

3.-e 启动Email参数，并配合-n选项来隐藏工具在爬取网站过程中生成的密码字典，如 -n -e，结果会只显示Email地址

4.-c 计算目标网站中某个词的重复出现次数，这样在生成的密码字典后面会有次数

5.-d 指定爬取深度，默认爬取深度为2

6.--debug 开启调试模式，可以查看网站爬取过程中出现的错误和元数据

7.-v 进入verbose模式，在该模式下，cewl会导出目标网站的详细数据

8.--with-numbers 生成包含数据和字符的字典文件

9.基础认证
    
    如果目标网站需要进行页面登陆认证的话，需要使用下列参数绕过页面认证的限制

    --auth_type Digest or basic

    --auth_user Authentication username

    --auth_pass Authentication password

    如 cewl http://ip --auth_type Digest --auth_user admin --auth_pass password -v

10.代理URL

    如果目标网站设置了代理服务器的话，Cewl将无法使用默认命令来生成字典，需要使用--proxy option选项来启动代理URL功能

    如 cewl --proxy_host 192.168.1.103 --proxy_port 3111 http://ip
```
