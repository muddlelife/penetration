# BurpSuite

BurpSuite 是用于Web应用安全测试工具的集成平台，它包含许多工具，并为这些工具设计了许多接口，以促进加快测试应用程序的过程

所有的工具都共享一个能处理并显示HTTP消息、持久性、认证、代理、日志、警报的一个强大的可扩展的框架

## 基本工具

1. Proxy
    
    一个拦截HTTP/S的代理服务器，作为一个在浏览器和目标应用程序之间的中间人，允许拦截、查看、修改在两个方向上的数据包

2. Spider
    
    一个应用智能感应的网络爬虫，它能完整地枚举应用程序的内容和功能

3. Scanner
    
    一个高级工具，执行后，能自动发现Web应用程序的安全漏洞

4. Intruder

    是一个定制的高度可配置的工具，对Web应用程序进行自动化攻击，如:枚举标识符、表单破解和信息收集

    Attack Type

    * Sniper:对变量依次进行破解，只有一个字典，参数有两个，先保持参数一不变，改变参数二，参数二遍历字典，然后让参数二不变，参数一改变，参数一遍历字典

    * Battering:ram:对变量同时进行破解，只有一个字典，参数两个，用户名和密码同时从字典遍历，而用户名和密码是相同的

    *   Pitch fork:每个变量对应一个字典，两个字典，两个参数，两个字典分别一一对应的行进行匹配，直到哪一个字典的行数先完成匹配，则结束

    *   Cluster bomb:每个变量将会对应一个字典，并且交集破解，尝试每一个组合，两个字典，两个参数，交叉匹配，保持一个参数的字典值不变，再去遍历另一个字典的所有值，遍历完后再让那个不变的参数的字典值更新一个，再去同样的操作，直至结束

5. Repeater

    是一个靠手动操作来补发单独的HTTP请求，并分析应用程序响应的工具

6. Sequencer

    是一个用来分析那些不可预知的应用程序会话令牌和重要数据项的随机性工具

7. Decoder

    极为方便的解码/编码工具

8. Comparer
    
    一个实用的工具，通常是通过一些相关的请求和响应得到两项数据的一个可视化的“差异”
